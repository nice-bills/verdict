# Verdict UI

Next.js frontend for Somnia testnet markets. Config defaults from `config/shannon.json` (synced from `../deployments/shannon.json` on `pnpm build`).

```bash
pnpm install && pnpm dev
```

Optional overrides: copy `.env.example` → `.env.local`.

**Deploy:** [../docs/HOSTING.md](../docs/HOSTING.md) (Vercel or GitHub Pages).

Design: [Hallmark](https://github.com/nutlope/hallmark) — see `design.md`.
