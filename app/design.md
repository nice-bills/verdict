# Verdict UI — studied DNA (reference video)

**Source:** user reference video (cinematic landing, not terminal/hacker).

## Locked system

- **Canvas:** full-viewport dark cinematic background (looping ambient video + dark scrim).
- **Type:** Instrument Serif display; Inter body/UI. No monospace in primary UI.
- **Layout:** center-aligned hero; generous vertical rhythm; transparent top nav (logo | links | CTA).
- **Surfaces:** liquid glass only — gradient fill, inset highlight, soft shadow, `backdrop-filter: blur(12px)`.
- **Buttons:** pill, glass (no solid orange fills); hover = `scale(1.03)`.
- **Motion:** staggered fade-rise (24px), 200ms ease-out; respect `prefers-reduced-motion`.
- **Banned:** matrix green, terminal chrome, heavy bordered cards, dashboard two-column hero, mono address chips.

## Tokens

See `tokens.css` — all colors via `--color-*`, fonts via `--font-*`.
